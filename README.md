# 💊 Сортировщик медпрепаратов

Веб-приложение для подбора медицинских препаратов по симптомам и параметрам пациента.

## Возможности

- 🔍 Интеллектуальный поиск препаратов по симптомам
- 👤 Учет индивидуальных параметров пациента (возраст, вес, рост, пол)
- ⚠️ Автоматическая фильтрация по противопоказаниям
- 🧬 Проверка на аллергии и хронические заболевания
- 📊 Ранжирование результатов по релевантности
- 📚 Полный каталог препаратов с инструкциями
- 💾 База данных SQLite с историей поиска
- 📈 Статистика использования системы

## Технологии

- Backend: Node.js + Express + Better-SQLite3
- Frontend: Vanilla JavaScript (ES6 modules)
- База данных: SQLite

## Установка и запуск

### 1. Установка зависимостей

```bash
cd backend
npm install
```

### 2. Запуск сервера

```bash
npm start
```

Сервер запустится на http://localhost:3000

### 3. Открытие приложения

Откройте файл `frontend/index.html` в браузере или используйте Live Server.

## Использование

1. Заполните данные пациента (пол, возраст, вес, рост)
2. Укажите хронические заболевания и аллергии (если есть)
3. Опишите симптомы
4. Нажмите "Найти препараты"
5. Просмотрите результаты и нажмите на препарат для подробной информации

## API Endpoints

- `GET /api/medications` - Получить все препараты
- `GET /api/medications/:id` - Получить препарат по ID
- `POST /api/patients` - Создать профиль пациента
- `POST /api/search` - Поиск препаратов
- `GET /api/stats` - Статистика системы

## Особенности алгоритма

- Автоматическая фильтрация по возрасту пациента
- Проверка противопоказаний на основе хронических заболеваний
- Проверка аллергических реакций на действующие вещества
- Ранжирование по релевантности симптомам (показания, ключевые слова, название)
- Сохранение истории поиска для анализа

## Предупреждение

⚠️ Приложение носит исключительно информационный характер. Перед применением любых препаратов необходимо проконсультироваться с врачом.
